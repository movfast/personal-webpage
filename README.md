# Marian Helcl Øverli — Personal Portfolio

Personal portfolio website showcasing AI/ML projects, blog writing, and an interactive AI twin chatbot.

**Live site:** [coming soon]

## Tech Stack

- Vanilla HTML, CSS, JavaScript — no frameworks or build tools
- Fully responsive with mobile-first design
- Dark theme with optional botanical light theme

## Features

- **Project showcase** with screenshots, tags, and links
- **Blog/writing section** with full article pages
- **Live AI news feed** from TechCrunch, The Verge, Ars Technica, and Google AI Blog
- **AI Twin chat** — Gradio-powered chatbot so visitors can ask questions about me
- **Easter egg system** — 13 hidden quirks with XP scoreboard (try the Konami code!)
- **Botanical theme** — toggle a nature-inspired light theme with animated SVG branches and floating petals

## Local Development

Just open `index.html` in a browser — no build step needed.

For live reload during development:

```bash
# Using Python
python3 -m http.server 8000

# Using Node
npx serve .
```

## Deployment

Hosted on GitHub Pages with a custom domain.
