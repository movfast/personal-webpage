Meow Master! — Cat Training Simulator

A browser-based cat training game powered by Bayesian Thompson Sampling. Watch real machine learning happen in real-time as you train a procedurally-drawn cat to perform tricks.

JavaScript Canvas 2D Web Audio API Thompson Sampling Bayesian ML

How to Play

You observe a cat performing random behaviors. Press Space (or click the cat) when you see the target trick. A Bayesian Multi-Armed Bandit model learns which behaviors you reward and offers them more frequently. Once the model is confident enough, you name the trick and advance to the next room.

InputAction
Space / Click catReward current behavior (costs 1 treat)
Click elsewhereDirect cat to walk there
EscPause / Resume
Click "Pet me!" bubbleCalm a frustrated cat

The ML Behind It

Each behavior is an "arm" in a Multi-Armed Bandit with a Beta(alpha, beta) distribution. Rewarding the target behavior increases alpha (the model learns to offer it more), while unrewarded behaviors slowly increase beta (offered less over time). Thompson Sampling draws from each arm's Beta distribution, then softmax picks the next behavior. The probability bars in the top-left corner visualize this learning in real-time.

Progression

7 rooms, each teaching a new trick — from Sit in the Cozy Living Room to Purr in the Enchanted Garden. Earn ranks from Beginner all the way up to Cat Whisperer.
