Draw & Classify — Neural Network Demo

Draw digits on the canvas and watch a 2-layer neural network classify them in real-time, with animated neuron activations and live probability bars.

JavaScript Canvas 2D Neural Network MNIST

How It Works

Draw a digit (0-9) on the canvas. Your drawing is downsampled to 28x28 pixels (like MNIST) and fed through a 784→64→10 neural network running entirely in JavaScript. The network diagram shows neuron activations in real-time.

ControlAction
Draw on canvasSketch a digit with mouse or touch

Gallery digitsLoad a pre-drawn example

The Network Architecture

The network has an input layer (784 neurons for the 28x28 image), a hidden layer (64 neurons with ReLU activation and hand-crafted spatial pattern detectors), and an output layer (10 neurons with softmax for digit probabilities). The visualization shows the most active hidden neurons glowing proportionally to their activation values.
